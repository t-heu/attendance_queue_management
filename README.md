# Tecnologias usadas
- express
- ejs (template engine)
- firebase

# Como executar
- crie o arquivo `firebaseConfig.js` em `/public/assets/js` e set suas configurações do firebase, e set também suas configurações no `.env`
```shell
yarn
yarn dev or yarn start
```
